# Raspberry-Pi-Motion-Detection-Camera-System
Developing a real-time motion detection system using a Raspberry Pi camera module and OpenCV, implementing frame differencing algorithms to identify and log motion events.

## Setup

1. Clone the repository
2. Create a virtual environment:
```
   python -m venv .venv
```
3. Activate it:
   - Windows (PowerShell): `.venv\Scripts\activate`
   - Windows (Git Bash): `source .venv/Scripts/activate`
   - Mac/Linux: `source .venv/bin/activate`
